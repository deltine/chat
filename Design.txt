・プロジェクト名
chat

・システム要件
チャットルームは一つ
マッチングは行わない
UIはラインのようなグループchat
 - 左側に相手のメッセージ
   右側に自分のメッセージ
サーバー切断時にログアウト
 - ホーム画面へは戻れない

・画面設計
トップ-ユーザーログイン画面
 - Output タイトル、使用しているパッケージ名
 - Input ユーザー名
 - Event ユーザー名登録、ログイン処理、チャット画面へ遷移

チャット画面
 - Output ログインしてユーザー名表示
 - Input メッセージ
 - Event メッセージ送信

・操作フロー
 - トップ画面
   ユーザー名を入力し確定する
 - チェット画面
   チャットルームへ参加の通知を行う
